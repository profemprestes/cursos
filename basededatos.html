1. Fundamentos de Bases de Datos

1.1. Conceptos Básicos

1.1.1. Definición y Tipos de Bases de Datos

Una base de datos es un conjunto organizado de datos estructurados, almacenados sistemáticamente para su posterior uso. El objetivo principal de una base de datos es proporcionar una forma eficiente de almacenar, recuperar y gestionar datos.

Las bases de datos se pueden clasificar en varios tipos según su modelo de datos:

  • Bases de datos relacionales: Organizan los datos en tablas relacionadas mediante claves. Ejemplos: MySQL, PostgreSQL, Oracle.
  • Bases de datos NoSQL: Diseñadas para modelos de datos específicos con esquemas flexibles. Ejemplos: MongoDB, Cassandra, Redis.
  • Bases de datos orientadas a objetos: Almacenan datos como objetos. Ejemplos: db4o, ObjectDB.
  • Bases de datos jerárquicas: Organizan los datos en una estructura similar a un árbol. Ejemplo: IBM IMS.
  • Bases de datos de red: Permiten relaciones muchos a muchos. Ejemplo: IDMS.
